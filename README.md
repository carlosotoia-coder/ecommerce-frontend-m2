# E-commerce Frontend - Módulo 2

Proyecto frontend correspondiente al Módulo 2 del curso, que consiste en el desarrollo de un MVP de una tienda en línea utilizando HTML5 semántico, Bootstrap y JavaScript básico.

## 📌 Descripción
Este proyecto simula un e-commerce con las siguientes funcionalidades:
- Página de inicio con grilla de productos
- Página de detalle de producto
- Carrito de compras simulado
- Contador de productos en el navbar
- Diseño responsivo (mobile-first)

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap 5 (CDN)
- JavaScript (DOM y eventos)
- Git y GitHub
